## How to Take Screenshots

### Windows:
- Press `Win + Shift + S` to open the screenshot tool
- Select the area you want to capture
- Paste into an image editor and save

### Or use:
- `Print Screen` key to capture full screen
- `Alt + Print Screen` to capture active window
